# AI Analytics - Exam Quality Analyzer

- **Сұрақтарды тексеру** - силлабусқа сәйкестікті тексеру
- **Сапа анализі** - сұрақтардың қиындығы мен дискриминативтілігінің анализі
- **Корреляция анализі** - сұрақтар мен жалпы баллдар арасындағы корреляция
- **Студенттердің үлгілері** - студенттердің оқу нәтижелерінің анализі
- **Семантикалық анализ** - қайталануларды семантикалық іздеу
- **Критерийлер анализі** - бағалау критерийлері мен өнімділіктің анализі
- **Студенттердің жауаптары** - студенттердің жауаптары мен плагиаттың анализі
- **Деректерді импорттау** - силлабустар мен емтихан нәтижелерін жүктеу

## Docker арқылы іске қосу

```bash
docker-compose up -d
```

- **app** - PHP қолданбасы (http://localhost:8080)
- **db** - MySQL 8.0 (порт 3307)
- **embeddings** - Эмбеддингтерге арналған Python API (порт 8000)


## Жергілікті орнату

- **PHP:** 8.0 +
- **MySQL:** 5.7 +
- **Веб-сервер:** Apache (XAMPP)
- **Python:** 3.8+
- **PHP кеңейтімдері:** pdo_mysql, mbstring, json, zip, gd

### Баптау

1. http://localhost/phpmyadmin ашыңыз
2. `exam_analyzer_2` атымен жаңа деректер қорын құрыңыз
3. SQL импорттаңыз: `sql/full_database_setup.sql`
4. Деректер қорына қосылуымды `ai_standalone/config.php` файлында баптаңыз
5. Бірінші іске қосқанда әдепкі пайдаланушы құрылады:
   - Логин: `superadmin`
   - Құпия сөз: `superadmin123`

### Python қызметі

Эмбеддингтер, семантика және силлабус импорты үшін Hugging Face моделі қолданылады:
- Модель: `sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2`
- Қолдау: орыс, ағылшын, қазақ тілдері

```bash
cd ai_standalone/python_services
pip install -r requirements.txt
python embeddings_api.py
```
