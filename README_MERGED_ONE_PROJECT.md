# Exam Quality Analyzer — единый проект

Полностью объединённая версия:
- база MySQL;
- интерфейс на PHP;
- предзагруженные демонстрационные данные из ODS;
- модуль анализа качества вопросов;
- AI-логика: силлабус, семантическое соответствие, правила классификации;
- пользователи и роли.

## Главное
Это не две папки рядом, а один запускаемый Open Server-проект.

Открывается через:
`http://site.loc/`

## SQL
Основной SQL:
`sql/exam_analyzer_full_install.sql`

Но при первом открытии сайт сам создаёт БД и загружает seed.sql.
