РЕАЛЬНЫЙ ИМПОРТ ODS/CSV

В этой версии import.php уже не заглушка.
Он реально:
- читает ODS через ZipArchive + content.xml;
- читает CSV через fgetcsv;
- проверяет русские заголовки;
- создаёт дисциплины, студентов, темы, вопросы;
- добавляет raw_exam_results;
- пересчитывает question_metrics;
- пересобирает данные для AI-аналитики.

Если ODS не импортируется:
1. Проверь, включён ли ZipArchive в XAMPP.
2. Если ZipArchive выключен, сохрани ODS как CSV и загрузи CSV.
